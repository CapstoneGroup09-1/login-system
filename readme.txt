简单配置操作手册

第一步：安装并解压WAMP环境

首先，你需要下载并安装WAMP环境。WAMP是一个集成了Web服务器（Apache）、数据库（MySQL）和PHP解释器的软件环境，非常适合进行网站开发。下载并安装好WAMP后，你会得到一个压缩包，解压这个压缩包，并将解压后的文件夹放在C:\wamp64\www目录下。

第二步：放置login文件

接下来，将你的login文件夹（包括其内部的所有文件）移动到C:\wamp64\www目录下。这样，你的login项目就放置在了WAMP服务器的根目录下，可以通过Web服务器来访问了。

第三步：导入数据库

在你的login文件夹中，有一个名为lianxi.sql的数据库文件。这个文件包含了你的网站所需要的数据库结构和数据。你需要将这个数据库文件导入到WAMP的MySQL数据库中。打开WAMP的管理界面，找到MySQL数据库的导入功能，选择lianxi.sql文件进行导入。

第四步：修改数据库配置

数据库导入成功后，你需要修改login项目中的数据库配置信息。打开C:\wamp64\www\login\php\config.php文件，找到数据库密码的部分（通常是pass），将默认的密码修改为你自己数据库的密码。如果你的数据库密码为空，就填写root；用户名（username）通常也是root。

第五步：访问登录页面

配置完成后，你就可以通过浏览器来访问你的登录页面了。在浏览器的地址栏中输入http://localhost:8080/login/login.html。这里的localhost指的是你的本机地址，8080是WAMP服务器的端口号。如果你在安装WAMP时修改了端口号，那么这里就需要填写你自己设置的端口号。

注意事项

确保WAMP服务已经启动，并且MySQL数据库也在正常运行。
